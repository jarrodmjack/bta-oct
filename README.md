## Getting Started


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jarrodmjack/bta-oct.git
   ```
2. Install NPM packages
   ```sh
   cd 02_front/frontend
   npm install
   
3. To get the Frontend running while in /frontend
   ```sh
   npm run dev
   ```
